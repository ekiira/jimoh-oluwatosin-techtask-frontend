# Frontend Tech Task

## Running the App Locally

```

# clone the repo
git clone https://github.com/ekiira/jimoh-oluwatosin-techtask-frontend.git

# cd to jimoh-oluwatosin-techtask-frontend
cd jimoh-oluwatosin-techtask-frontend/

# Install Dependencies
yarn

# Run the application
yarn start

```

## Test

```
# run unit tests
yarn test

```
